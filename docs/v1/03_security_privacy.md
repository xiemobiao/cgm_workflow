# Security and Privacy (V1)

## Authentication
- JWT for API access
- Optional SSO integration later

## Authorization
- Project-level RBAC
- Gate approval restricted to QA/Release/Admin

## Secrets
- Store integration tokens encrypted
- Rotate credentials regularly

## Privacy
- Avoid storing PII in logs
- Mask device identifiers if required by policy

## Audit
- Record gate approvals, overrides, and integration changes
