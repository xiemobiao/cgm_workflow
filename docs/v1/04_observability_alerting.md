# Observability and Alerting (V1)

## Metrics
- API latency, error rate, request volume
- Queue depth and worker lag
- Log ingestion rate and parse failures

## Logs
- API access logs
- Worker processing logs

## Alerts
- API error rate > threshold
- Queue backlog > threshold
- Log parse failure rate > threshold

## Dashboards
- Workflow throughput
- Incident counts by severity
- Log ingestion stats
